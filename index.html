Blox

Blox is a Component Library project. It's built by and for Wax Interactive, but if you find it useful, feel free to clone, fork or report issues.

Install

From the command line:

  1. Install grunt-cli globally with npm install -g grunt-cli.
  2. Navigate to the root /blox directory, then run npm install. npm will look at package.json and automatically install the necessary local dependencies listed there.
  3. Navigate to the root /blox directory, then run bower install. bower will look at bower.json and automatically install the necessary libraries listed there.

When completed, you'll be able to run the various Grunt commands provided from the command line.
